MATErix: Cold, hard facts provided by a warm, loving friend

MATErix is a command-line utility for solving computational linear algebra problems.

View on Github

MATErix equates with simplicity and speed

MATErix is a script written in Python 3 and runnable directly from the terminal. That means no installation overhead and no confusing multi-window user interface - just all the functionality you need packaged in a fast, lightweight application.

Written with the user, not the developer, in mind

MATErix does away with Developer Ego Syndrome and works with the user's system to provide a simple, yet intuitive interface. MATErix uses text files as its input medium, ensuring that every user has the machinery necessary to run MATErix without the need for any installations (except Python).

An Integrated Mathematics Toolkit

MATErix supports operations essential to linear algebra. Syntax and style are streamlined across the different program domains, and the entire user experience is tightly integrated regardless of intended application.

Written in Python

MATErix is written completely in Python and doesn't make use of external libraries or APIs. Python is a powerful, open-source programming language known for its ubiquity across platforms and applications.

Flexible and Extensible

MATErix stores all input, output, and performance data as .txt files allowing you the opportunity to perform data post-processing, provide input to another program, or archive results for later use. What's more, MATErix boasts a clean, readable API that can be readily extended to add custom functionality.

Advanced Algorithm Implementations
Because sometimes you need to pull out the big guns.

MATErix supports the Strassen algorithm for matrix multiplication. While the default iterative algorithm will suffice in most cases, the strassen algorithm provides huge performance gains when multiplying matrices of large dimensions. Support for algorithms utilizing advanced techniques such as multi-threading are slated for future releases.

Performance Monitoring
For those instances when speed is everything.

MATErix ships standard with tools for monitoring its runtime performance. These allow you to determine the most beneficial MATErix settings for your unique needs. It's the sort of meta-analysis your high-school English teacher would be proud of.
